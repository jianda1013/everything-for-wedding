This project is written to host a web page for monogram

# Base structure

## Intros

this is a book like web application. user can choose to start read the book from the start or end.
if read the book from the beginning, the user can read the story of Nina. and also read the story of Jianda's if read it backwards.

user will keep turn the page to read the book. and the story will meet together when Jianda and Nina were met.

After that the story combine together into one.

## Tech target
- react typescript with vite base
- book styled webpage
- user can turn the page
- user can choose which way to read the book (forwards or backwards)
- png image display for books.
- png image display for each object in a page (can determine the position with axis)


## Deploy goal

- can be serve as static page on gcp
- using npm for package manager
- git register on Github
